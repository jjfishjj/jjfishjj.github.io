Project Purpose

規劃本地語意搜尋與文件問答系統，展示 retrieval、metadata、權限與可追溯答案設計。

Target Scenario

法律科技公司在本地部署 NeMo Retriever，索引 50 萬份合約做語意搜尋問答。

Proposed Architecture

  1. 定義 input data、model、asset、document 或 video stream。
  2. 建立 baseline workflow 與假設條件。
  3. 導入對應 NVIDIA skill 作為 acceleration、training、deployment 或 simulation layer。
  4. 用 benchmark、demo walkthrough 或 checklist 驗證結果。
  5. 整理成 GitHub README、demo page 與 LinkedIn 摘要。

Deliverables

  • 文件索引設計
  • metadata schema
  • 查詢結果格式
  • 可追溯引用範例

Milestones

  1. 設計文件分類
  2. 建立 metadata schema
  3. 規劃查詢與排序
  4. 整理展示案例
