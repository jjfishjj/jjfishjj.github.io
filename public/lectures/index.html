Chunking Method

團塊法

預計公開

把零散詞彙、句型與發音線索整理成可記憶的語言團塊， 降低大腦負擔，讓輸入更容易變成可提取的表達。

  • 建立語言團塊分類表
  • 練習短句到長句的組合
  • 設計自己的記憶回收流程
預計形式 90 分鐘講座 + 練習工作紙

Spelling Memory System

拼寫記憶術

規劃中

從字母形狀、音節節奏、語源線索與常見拼寫模式切入， 建立更穩定的拼寫記憶，不只背單字，也理解單字如何被組成。

  • 拆解容易混淆的拼寫
  • 建立音形連結
  • 設計跨語言拼寫提醒卡
預計形式 75 分鐘講座 + 示範練習
