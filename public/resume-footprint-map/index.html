我的履歷路線

2009-2011

交通大學 / Solid State Lab

我研究碳摻雜 TiO2 薄膜的電性、光學、磁性與材料缺陷，並把材料分析經驗連到半導體薄膜製程、晶片感測元件與 IC power efficiency 的理解。

  • XANES、XRD、ESCA、SQUID 等材料與磁性分析經驗
  • 薄膜製程、缺陷結構、dipole spin 與感測應用方向
  • 對 CPU / GPU / TPU 所需材料、製程與可靠性問題建立底層語言
Hsinchu, Taiwan
2014-2018

Huiria / 電子零組件與採購經驗

我接觸電子零組件、晶圓元件、CPU、GPU、主機板與供應鏈溝通，也參與展會與跨區採購協調，理解硬體產品從供應、成本到交付的現實限制。

  • 採購、供應商協調、參展與代理商溝通
